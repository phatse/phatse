### VSTEP Exam Practice Platform

A VSTEP English learning and exam practice platform with learner, admin, teacher, and payment workflows.

**My focus areas:**

- React learner/admin UI flows for practice, mock exams, result review, wallet/course enrollment, content CRUD, finance dashboards, booking schedules, and grading rubrics
- API integration with a Laravel 13 backend using PostgreSQL, Redis, Horizon queues, Octane, JWT auth, and Cloudflare R2/S3-compatible audio storage
- PayOS payment return/callback flows, wallet ledger credits/refunds, password reset, email verification, and Resend email integration
- AI-assisted Writing/Speaking grading UX with progress states, diagnostics, result pages, paid feedback, and teacher review requests
- Debugging edge cases such as duplicate payments, expired exam auto-submit, stale Octane reads, profile isolation, booking conflicts, and cache invalidation

**Scale of the codebase:** 356 API endpoints, 127 migrations, and 86 backend tests.

Repository is private. Code walkthrough or demo can be provided upon request.

### DEXIN Interior Design & Consultation Platform

Frontend development for an AI-powered interior design and consultation platform.
